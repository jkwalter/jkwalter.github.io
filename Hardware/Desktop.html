Custom Desktop Computer

About

I built my custom desktop computer with help from a friend in April 2014. Although this computer was built for the purpose of playing video games, I also use it for programming, everyday computing, and other entertainment purposes. Shopping for the components and building the computer sparked my interest in computer hardware, and I still enjoy looking at current computer hardware components and specifications. I have upgraded some components since the computer was built, but most of the original components remain. Please feel free to read the Specifications section for more information about the current components in my computer.
